## ... als Herrschaftsform
+ die Macht wird direkt vom Volk in Abstimmungen ausgeübt
+ kein Repräsentativsystem
+ Basisdemokratie, Anarchie
## ... als politisches Entscheidungsverfahren
+ Teil einer repräsentativen Demokratie
+ Stimmberechtigte stimmen unmittelbar über Sachfragen ab
+ "halbdirekte" oder "plebiszitäre Demokratie"
# Instrumente der direkten Demokratie
1. Referenden (Sg.: Referendum)
	+ direkte Abstimmung des Volkes
	+ abgestimmt wird über eine Abstimmungsvorlage, die von einer gewählten Vertretung (Parlament) stammt
2. Initiativverfahren
	+ das Volk ergreift die Initiative
	+ das Volk bringt ein Anliegen oder eine Vorlage zur zwingenden Beratung im Parlament ein
	+ Abstimmung über Annahme oder Ablehnung erfolgt durch das Parlament
3. Volksabstimmungsverfahren
	+ das Volk erarbeitet eine Vorlage zur Abstimmung
	+ Abstimmung erfolgt durch das Volk
# direkte Demokratie in Deutschland
1. auf Bundesebene:
	+ zur Zeit kein Initiativrecht für das Volk
2. auf Landesebene; Beispiel Baden-Württemberg:
	+ Volksantrag: der Landtag muss sich mit dem Antrag befassen, wenn ein Quorum von 0,5% der Wahlberechtigten innerhalb eines Jahres erreicht ist
	+ Volksbegehren: ein ausgearbeiteter Gesetzentwurf wird dem Landtag zur Abstimmung vorgelegt
	+ Volksabstimmung: entweder Abstimmung über eine Gesetzesvorlage, die durch Volksbegehren eingebracht wurde; oder auf Verlangen eines Drittels der Mitglieder des Landtags (Zustimmungsquorum: 20% der Stimmberechtigten)
